Structure Boilerplate

This is a standard framework for developing web apps.

Pattern
.
/-> assets
|    /-> coffee
|    |    |-> app.coffee
|    |
|    /-> css
|    |    /-> fonts
|    |    |-> fonts.css
|    |    |-> footer.css
|    |    |-> header.css
|    |    |-> init.css
|    |    |-> main.css
|    |
|    /-> img
|    |    |-> favicon.ico
|    |
|    /-> js
|    |    /-> lib
|    |    |    |-> app.js
|    |    |
|    |    /-> vendor
|    |         |-> analytics.js
|    |         |-> html5.js
|    |         |-> jquery.js
|    |
|    /-> sass
|        |-> config.rb
|        |-> fonts.sass
|        |-> footer.sass
|        |-> header.sass
|        |-> init.sass
|        |-> main.sass
|
/-> inc
|    |-> controller.php
|    |-> footer.php
|    |-> header.php
|    |-> send-contact.php
|    |-> smtp.php
|
/-> pages
|    |-> home.php
|
|-> .htaccess
|-> README.md (this)
|-> index.php
